# Explosion

An **Explosion** is a visual effect displayed at an [Enemy](Enemy.md)'s position briefly when the [Enemy](Enemy.md) is destroyed by a [Bullet](Bullet.md).

## Requirements

1. When an [Enemy](Enemy.md) is destroyed by a [Bullet](Bullet.md), an **Explosion** must be displayed at the [Enemy](Enemy.md)'s position.
2. The **Explosion** must be displayed briefly, then removed automatically.
3. The **Explosion** must not affect gameplay — it is a purely visual effect.
